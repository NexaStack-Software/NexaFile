# NexaSign einfach erklärt

Diese Anleitung ist für Menschen, die NexaSign zum ersten Mal sehen.
Sie müssen nichts Technisches wissen.

Es geht nur um eine Frage:

**Was kann ich mit NexaSign machen?**

---

## Kurz gesagt

Mit NexaSign können Sie Dokumente digital unterschreiben lassen.

Zum Beispiel:

- ein NDA
- einen Vertrag mit einem Kunden
- einen Arbeitsvertrag
- eine Auftragsbestätigung
- einen AV-Vertrag nach DSGVO
- eine Vereinbarung mit einem Dienstleister
- ein internes Freigabe-Dokument

Sie laden ein PDF hoch, sagen, wer unterschreiben soll, setzen die Felder ins
Dokument und versenden den Vorgang.

Danach sehen Sie jederzeit, ob das Dokument noch offen ist oder bereits
abgeschlossen wurde.

---

## Wofür ist NexaSign gut?

NexaSign hilft Ihnen, wenn Sie sonst Dokumente per E-Mail hin und her schicken,
ausdrucken, unterschreiben, einscannen und wieder zurücksenden würden.

Mit NexaSign läuft das einfacher:

1. Dokument hochladen.
2. Empfänger eintragen.
3. Unterschriftsfeld setzen.
4. Senden.
5. Warten, bis unterschrieben wurde.
6. Fertiges Dokument herunterladen oder später wiederfinden.

Das ist der Kern.

---

## Was sehe ich zuerst?

Nach dem Login landen Sie in Ihrer Dokumentenübersicht.

Dort sehen Sie Ihre Arbeitsliste:

- Entwürfe
- offene Vorgänge
- abgeschlossene Dokumente
- Dokumente, die noch bearbeitet werden müssen

Wenn Sie nicht wissen, wo Sie anfangen sollen: **Starten Sie immer bei
„Dokumente".**

---

## Ein Dokument unterschreiben lassen

So läuft ein typischer Vorgang:

### 1. PDF vorbereiten

Sie brauchen ein PDF.

Das kann ein Vertrag, ein Angebot, eine Vereinbarung oder ein anderes Dokument
sein.

### 2. PDF hochladen

Klicken Sie in NexaSign auf **Upload Document**.

Wählen Sie das PDF von Ihrem Computer aus.

### 3. Empfänger eintragen

Tragen Sie ein, wer das Dokument unterschreiben soll.

Das kann eine Person sein oder mehrere Personen.

Zum Beispiel:

- Kunde
- Mitarbeiterin
- Geschäftsführer
- Dienstleister

### 4. Felder setzen

Jetzt legen Sie fest, wo im Dokument unterschrieben werden soll.

Sie können zum Beispiel ein Feld für die Unterschrift setzen.

Je nach Dokument können auch weitere Felder sinnvoll sein, zum Beispiel Name
oder Datum.

### 5. Dokument senden

Wenn alles passt, senden Sie das Dokument ab.

Die Empfänger bekommen einen Link und können das Dokument öffnen und
unterschreiben.

### 6. Status verfolgen

In der Dokumentenübersicht sehen Sie, was passiert ist:

- **Draft**: noch ein Entwurf
- **Pending**: wurde versendet, wartet noch auf Unterschrift
- **Completed**: ist fertig unterschrieben

So sehen Sie sofort, wo noch etwas offen ist.

---

## Vorlagen nutzen

NexaSign enthält deutsche Vorlagen und Generatoren.

Das ist hilfreich, wenn Sie noch kein fertiges PDF haben.

Sie können zum Beispiel:

- ein NDA vorbereiten
- einen Freelancer-Vertrag erstellen
- einen Arbeitsvertrag vorbereiten
- einen AV-Vertrag nach DSGVO erzeugen
- eine X-Rechnung / ZUGFeRD-Datei vorbereiten

Der Ablauf ist einfach:

1. Öffnen Sie **Vorlagen**.
2. Wählen Sie eine passende Vorlage.
3. Füllen Sie die Felder aus.
4. Erzeugen Sie ein PDF.
5. Laden Sie dieses PDF in NexaSign hoch.
6. Lassen Sie es unterschreiben.

So kommen Sie von „Ich brauche ein Dokument" zu „Das Dokument ist fertig und
kann unterschrieben werden".

## Was kann ich in der Demo ausprobieren?

Wenn Sie NexaSign nur ansehen möchten, starten Sie am besten mit diesen Dingen:

### Dokumentenliste ansehen

Schauen Sie sich an, welche Dokumente offen, fertig oder noch Entwürfe sind.

### Tour starten

Klicken Sie auf **Tour starten**.

NexaSign zeigt Ihnen die wichtigsten Bereiche direkt im Bildschirm.

### Beispiel-Dokument öffnen

Öffnen Sie ein vorhandenes Dokument und schauen Sie, wie ein Vorgang aufgebaut
ist.

### Eigenes PDF hochladen

Laden Sie ein Test-PDF hoch und gehen Sie den Ablauf einmal durch.

### Vorlagen ansehen

Öffnen Sie den Bereich **Vorlagen** und prüfen Sie, welche Dokumente Sie direkt
erzeugen können.

## Typische Alltagssituationen

### „Ich brauche schnell ein unterschriebenes NDA."

Öffnen Sie die NDA-Vorlage, erzeugen Sie ein PDF, laden Sie es hoch und senden
Sie es an die andere Person.

### „Ein Kunde soll ein Angebot bestätigen."

Laden Sie das Angebot als PDF hoch, setzen Sie ein Unterschriftsfeld und senden
Sie es an den Kunden.

### „Ein neuer Mitarbeiter soll einen Vertrag unterschreiben."

Erzeugen oder laden Sie den Vertrag hoch, tragen Sie den Mitarbeiter als
Empfänger ein und verfolgen Sie den Status in NexaSign.

### „Ich will sehen, was noch offen ist."

Öffnen Sie die Dokumentenübersicht und filtern Sie nach offenen Vorgängen.

### „Ich brauche das fertige unterschriebene PDF."

Öffnen Sie das abgeschlossene Dokument und laden Sie es herunter.

## Was bedeutet „selbst gehostet" in einfacher Sprache?

Viele Signaturdienste laufen komplett bei einem fremden Anbieter.

NexaSign kann stattdessen auf der eigenen Infrastruktur betrieben werden.

Für Sie als Anwender bedeutet das:

- Ihre Organisation behält mehr Kontrolle über die Dokumente.
- Die Daten liegen nicht automatisch bei einem fremden Signaturdienst.
- Ihre IT oder Ihr Dienstleister kann NexaSign für Ihr Unternehmen betreiben.

Wenn Sie das intern vorschlagen möchten, reicht oft dieser Satz:

**„NexaSign ist eine Open-Source-Lösung, mit der wir Dokumente digital
unterschreiben lassen können, ohne dafür zwingend einen externen
Signatur-SaaS zu nutzen."**

---

## Was NexaSign nicht automatisch ersetzt

NexaSign macht den Unterschriftsprozess einfacher.

Es ersetzt aber nicht automatisch:

- rechtliche Prüfung eines Vertrags
- interne Freigaben
- Beratung durch Steuerberater, Anwalt oder Datenschutzbeauftragte
- besondere gesetzliche Anforderungen an bestimmte Dokumente

Wenn ein Dokument sehr wichtig ist oder besondere gesetzliche Formvorgaben hat,
sollten Sie vorher klären, welche Art von Signatur dafür nötig ist.

---

## Der einfachste Weg zum Verständnis

Wenn Sie NexaSign zum ersten Mal öffnen, machen Sie diese fünf Dinge:

1. Öffnen Sie **Dokumente**.
2. Klicken Sie auf **Tour starten**.
3. Öffnen Sie ein Beispiel-Dokument.
4. Laden Sie ein eigenes Test-PDF hoch.
5. Schauen Sie sich danach **Vorlagen** an.

Danach wissen Sie, wofür NexaSign da ist.

Nicht theoretisch, sondern praktisch.
